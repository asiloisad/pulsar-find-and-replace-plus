# find-and-replace-plus

## Changes

- Fixed project search and ripgrep on Windows.
- Fixed line ending normalization in buffer search.
- Added replace in-place functionality.
- Added center scroll when navigating from result view.
- Added buttons for ripgrep and PCRE2 regex options.
- Little reworked result view component.
- Converted CoffeeScript to JavaScript.
- Reduced dependencies.

## Installation

To install `find-and-replace-plus` search for [find-and-replace-plus](https://web.pulsar-edit.dev/packages/find-and-replace-plus) in the Install pane of the Pulsar settings or run `ppm install find-and-replace-plus`. Alternatively, you can run `ppm install asiloisad/pulsar-find-and-replace-plus` to install a package directly from the GitHub repository.
